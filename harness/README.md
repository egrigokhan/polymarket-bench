forecast-card protocol, and the pre-registered analysis plan. Frozen at season start
(`v1.0` tag); mid-season changes only for safety/integrity bugs, publicly changelogged.

See [PLAN.md §8](../PLAN.md) (harness), §6 (analysis plan), §4 (freeze rules).

- `analysis-plan.md` — pre-registered estimands, resampling units, checkpoints
- `CHANGELOG.md`
